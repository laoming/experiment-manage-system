# experiment-manage-system

## 项目介绍
这是一个实验管理系统的本科毕业设计。
项目在开发过程中引入了 CodeBuddy 作为辅助编码工具，是作者结合人工智能编码技术的一次实践尝试。
系统登录页为 http://localhost:8081/ems/pages/index.html

## （前期预设计）功能模块清单

1. 运营端（管理员）
   - 用户与权限管理
   - 通知与公告管理
   - 系统配置与监控
   - 数据统计与分析

2. 客户端（老师）
   - 课程与实验项目管理
   - 实验报告模板管理
   - 实验任务下发
   - 低代码实验报告模板生成
   - 报告评阅与反馈

3. 客户端（学生）
   - 实验项目查看与报告填写
   - 导入数据生成报告
   - 报告提交与暂存
   - 报告查询与下载
