# experiment-manage-system

## 项目介绍
这是一个实验管理系统的本科毕业设计。
项目在开发过程中引入了 CodeBuddy 作为辅助编码工具，是作者结合人工智能编码技术的一次实践尝试。
系统登录页为 http://localhost:8081/ems/pages/index.html

## （前期预设计）功能模块清单

1. 运营端（管理员）
   - 用户与权限管理
   - 通知与公告管理
   - 系统配置与监控
   - 数据统计与分析

2. 客户端（老师）
   - 课程与实验项目管理
   - 实验报告模板管理
   - 实验任务下发
   - 低代码实验报告模板生成
   - 报告评阅与反馈

3. 客户端（学生）
   - 实验项目查看与报告填写
   - 导入数据生成报告
   - 报告提交与暂存
   - 报告查询与下载

## 菜单列表

### （1）身份管理

1. 用户管理（管理员）
   - 新增、修改、删除用户
   - 一个用户必须关联一个角色
2. 组织管理（管理员）
   - 新增、修改、删除组织
3. 角色管理（管理员）
   - 新增、修改、删除角色
   - 角色下新增菜单
4. 菜单管理（管理员）
   - 新增、修改、删除菜单

### （2）课堂管理

1. 课程管理（管理员、老师）
   - 分页查询有哪些课堂
   - 新增、修改、删除课程
   - 课程和模板关联
   - 课程和学生关联
2. 我的课堂（学生）
   - 分页查询有哪些课堂
   - 点击课堂进去能查看有哪些实验任务
   - 点击实验任务可以根据模板提交实验报告

### （3）实验管理

1. 实验模板管理（管理员、老师）
   - 新增、修改、删除模板
2. 实验报告管理（管理员、老师）
3. 实验报告记录（学生）
4. 实验报告批阅（老师）
   - 打回实验报告，并提出修改意见
   - 给学生提交的实验报告评分

### （4）系统管理

1. 通知与公告（管理员、老师）
2. 系统配置与监控（管理员、老师）
3. 数据统计与分析（管理员、老师）
