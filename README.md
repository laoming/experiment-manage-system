# experiment-manage-system

## 项目介绍
这是一个实验管理系统的本科毕业设计。
项目在开发过程中引入了 CodeBuddy 作为辅助编码工具，是作者结合人工智能编码技术的一次实践尝试。
系统登录页为 http://localhost:8081/ems/pages/index.html

## （前期预设计）功能模块清单

1. 运营端（管理员）
   - 用户与权限管理
   - 通知与公告管理
   - 系统配置与监控
   - 数据统计与分析

2. 客户端（老师）
   - 课程与实验项目管理
   - 实验报告模板管理
   - 实验任务下发
   - 低代码实验报告模板生成
   - 报告评阅与反馈

3. 客户端（学生）
   - 实验项目查看与报告填写
   - 导入数据生成报告
   - 报告提交与暂存
   - 报告查询与下载

## 菜单列表

### （1）身份管理

1. 用户管理（管理员）
   - 提供用户列表分页查询，支持按条件筛选。
   - 支持用户创建，创建时必须关联一个组织和一个角色。
   - 支持用户信息编辑，可修改用户基本资料及关联关系。
   - 支持用户删除操作。

2. 组织管理（管理员）
   - 组织信息以组织树形式展示。
   - 提供 “展开全部” 功能，一键展开所有组织节点。
   - 支持组织新增。除根组织外，必须关联一个上级组织。
   - 支持组织信息修改。
   - 支持组织删除。删除前校验，组织下无用户方可删除。

3. 角色管理（管理员）
   - 提供角色列表分页查询，支持按条件筛选。
   - 支持角色新增、修改；删除时校验，已被使用的角色不可删除。
   - 支持角色与菜单列表关联配置，实现基于角色的菜单权限分配。

4. 菜单管理（管理员）
   - 提供菜单列表分页查询，支持按条件筛选。
   - 支持菜单新增、修改、删除。删除时校验，已被角色关联的菜单不可删除。

### （2）课堂管理

1. 课程管理（管理员、老师）
   - 支持课程列表分页查询。
   - 提供课程新增、修改、删除功能，维护课程基础信息。
   - 支持课程与学生关联。先选择组织，再选择学生，支持全选操作。

2. 我的课程（学生）
   - 支持课程分页查询。
   - 进入课程后，可查看该课程下所有实验任务。
   - 选择实验任务，可按模板填写并提交实验报告。

3. 我的授课（老师）
   - 支持所授课程列表分页查询。
   - 支持在课程下选择模板，新建实验任务。
   - 可点击课程，查看学生提交的实验报告，支持按条件筛选。
   - 支持实验报告批阅。可评分，也可打回并填写修改意见，打回后自动发送通知。
   - 支持向学生发送实验报告提交提醒。

### （3）实验管理

1. 实验模板管理（管理员、老师）
   - 支持实验模板列表分页查询，可按条件筛选。
   - 提供实验模板新增、修改、删除功能。

### （4）系统管理

1. 通知与公告（管理员、老师）
   - 支持公告发布与消息推送，可定向发送给指定用户/角色/组织。

2. 系统配置与监控（管理员、老师） 
   - 支持配置实验任务未完成自动提醒的时间阈值。

3. 数据统计与分析（管理员、老师）
   - 支持按实验任务统计学生得分情况。
   - 支持按课程统计单个学生的综合得分。
   - 支持按课程维度统计整体分数分布与完成情况。
